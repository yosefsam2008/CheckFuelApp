# עדכון חישוב יעילות תרמית - Thermal Efficiency Update

## מה השתנה?

עברנו מחישוב **בקפיצות של 10 שנים** לחישוב **שנתי רציף** של יעילות המנוע.

---

## 🔴 לפני (Old Method) - קפיצות של 10 שנים

```
2025: 35.0% ┐
2024: 35.0% │
2023: 35.0% │
2022: 35.0% │
2021: 35.0% │
2020: 35.0% ┘ ← קפיצה!
2019: 32.0% ┐
2018: 32.0% │
...
2010: 32.0% ┘ ← קפיצה!
2009: 30.0%
```

**בעיה**: רכבים מאותה תקופה (למשל 2019 ו-2020) מקבלים יעילות שונה לגמרי.

## 🟢 אחרי (New Method) - חישוב רציף שנה-שנה

### בנזין (Gasoline)
- **ירידה שנתית**: 0.3% לכל שנה אחורה
- **יעילות מקסימלית**: 36% (רכבים מ-2025 ואילך)
- **יעילות מינימלית**: 26% (רכבים ישנים מאוד)

```
2025: 36.00% →  0 שנים
2024: 35.70% → -0.30%
2023: 35.40% → -0.60%
2020: 34.50% → -1.50%
2015: 33.00% → -3.00%
2010: 31.50% → -4.50%
2005: 30.00% → -6.00%
2000: 28.50% → -7.50%
1990: 26.00% → מינימום
```

### דיזל (Diesel)
- **ירידה שנתית**: 0.25% לכל שנה אחורה (דיזל מתדרדר לאט יותר)
- **יעילות מקסימלית**: 42% (רכבים מ-2025 ואילך)
- **יעילות מינימלית**: 32% (רכבים ישנים מאוד)

```
2025: 42.00% →  0 שנים
```

## 📐 נוסחת החישוב

```javascript
יעילות = יעילות_מקסימלית - (שנים_אחורה × ירידה_לשנה)
יעילות = Math.max(יעילות_מחושבת, יעילות_מינימלית)
```

### דוגמה לבנזין 2015:
```
שנים_אחורה = 2025 - 2015 = 10 שנים
יעילות = 36% - (10 × 0.3%) = 36% - 3% = 33%
```

## 🎯 יתרונות השיטה החדשה

1. **צדק**: כל שנת ייצור מקבלת יעילות ייחודית
2. **רציפות**: אין קפיצות פתאומיות בין שנים
3. **דיוק**: חישוב מבוסס על ירידה ליניארית של טכנולוגיית מנועים
4. **גמישות**: קל לעדכן את הפרמטרים בעתיד
